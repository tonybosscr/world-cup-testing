# Mini App Dashboard

This is a lightweight premium-style dashboard for the World Cup prediction bot.

## Features
- Shows today's predictions
- Shows selected model
- Shows backtest stats
- Shows model comparison
- Button to show the current upcoming match prediction

## Data source
The app reads:
- `../output/dashboard_data.json`

## Preview note
In the workspace preview, local file loading may depend on file layout. When hosted in GitHub Pages or served from the same repo structure, it works as a static mini app.
